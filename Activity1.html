What is HTML?

  • HTML is the standard markup language for creating Web pages which stands for Hyper Text Markup Language


    Different Kinds of HTML tags

    HTML tags that can manipulate a text like size, font, appearance, etc...

    Here are some examples of HTML tags:

    1. Lets start with a basic '< p >' tag. This tag is a paragraph text which used when creating a paragraph, which we are using right now.

    2. Next is the '< i >' tag. → This tag shows a italic text which is used to manipulate a text by making a side like text.

    3. Next is the '< b >' tag. → This tag shows a bold text which is used to manipulate a text by making a text strong.

    4. Next is the '< center >' tag. → This tag can manipulate text by center-aligning a text, which we are using right now.

    5. Next is the '< big >' tag. → This tag shows to enlarge the text, which we are using right now.

    6. Next is the '< u >' tag. → This tag can manipulate text by underlining a paragraph or text, which we are using right now.

    7. Next is the '< mark >' tag. → This tag can manipulate text by highlighting a text, which we are using right now.

    8. Next is the '< li >' tag. ↓
